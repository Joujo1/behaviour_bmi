# pi_bmi_rig

A parallelized behavioral training rig for the Poisson click discrimination task, developed by Joel Kiener (jkiener@ethz.ch or joel.kiener@gmx.ch) as a Bachelor Thesis at the Neurotechnology Group, ETH Zürich (June 2026). The system runs multiple operant conditioning cages in parallel, each driven by a Raspberry Pi edge node, orchestrated from a central Linux PC with a web-based interface for session control, live monitoring, and curriculum management.
